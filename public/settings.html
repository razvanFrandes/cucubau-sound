How it works: All your recordings are saved as WAV files directly to your selected folder. The folder structure in the app matches the folder structure on disk. A cucubau-library.json file stores metadata like tags, ratings, and BPM.

Using with Ableton/DAWs: Use "Copy Path" in the app to copy the full file path, then paste it directly in your DAW's file browser or use Cmd+Shift+G (macOS) / Ctrl+Shift+G (Windows) to navigate to the file.

Note: After restarting your browser, you may need to grant permission again to access the folder.
